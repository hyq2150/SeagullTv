# 项目介绍：
	这是一款MaterialDesign设计风格视频播放器包含央视、卫视、地方、体育、卡通等高清直播以及搞笑视频。直播资源来自互联网，搞笑视频来自美拍，本应用只做学习使用，请勿商用！
